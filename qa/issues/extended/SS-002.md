# Reproducible Builds

**ID:** `SS-002`
**Severity:** MEDIUM
**Category:** supply-chain
**Reference:** Reproducible Builds

## Description

Builds must be reproducible: same source produces same binary hash.

## Verification Method

Build twice from clean state. SHA256 hashes must match.
